# OnPoint OpenAPI Viewer

Open the viewer and pass a spec name in the query string.

## Specs
- fleet-tenancy.yaml
- ingest.yaml
- trip-summary.yaml
- user-management.yaml

## View
Open index.html in a local static server, then use:
- index.html?spec=fleet-tenancy.yaml
- index.html?spec=ingest.yaml
- index.html?spec=trip-summary.yaml
- index.html?spec=user-management.yaml

If you use VS Code, the “Open with Live Server” action on index.html works well.
